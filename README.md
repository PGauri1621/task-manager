Backend:

Install all dependency via npm install.
To run the application - npm start.

Frontend:
Install all dependency via npm Install.
To run the application - npm run dev.

DB:
Create table with name task_manager and two columns users and tasks.

Tech stack:
Frontend - ReactJS, UI library - Material UI. State management - ContextAPI.

Backend - NodeJS 

DB - Postgresql (sql based)

The selection and decision of the tech stack is based on system requirements mentioned in the document.
